# Dangerous Debt

Openmw mod that puts new characters into debt. They need to deal with debt collectors.

## Installing

Extract to your `mods/` folder. In your `openmw.cfg` file, add these lines in the correct spots:

```ini
data="/wherevermymodsare/mods/ErnDebt-main"
content=ErnDebt.omwscripts
content=ErnDebt.omwaddon
```
